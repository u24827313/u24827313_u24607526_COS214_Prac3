\doxysubsection{Detailed Description}
Represents a user participating in a chat room. 

The \mbox{\hyperlink{classUsers}{Users}} class allows a user to send and receive messages within a \mbox{\hyperlink{classChatRoom}{Chat\+Room}} and manage a queue of commands that can be executed later.
